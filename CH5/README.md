1. Bernstein-Vazirani Algorithm

## Files

- `code-05-bernstein-vazirani.py`: Bernstein-Vazirani algorithm for finding the secret code (oracle) of a quantum padlock.
- `code-05-deutsch-jozsa.py`: Deutsch-Jozsa algorithm for checking if a quantum function is balanced or not.
- `code-05-quantum-teleportation.py`: Quantum teleportation for transmitting a qubit via a communication channel.
- `code-05-quantum-gate-teleportation.py`: Quantum teleportation for transmitting two entangled qubits via a communication channel.
- `code-05-superdense-coding.py`: Superdense coding for transmitting two classical bits via a single qubit.
- `code-05-fidelity-of-density-matrix.py`: Computing the fidelity of a density matrix for a noisy qubit.
- `code-05-fidelity-of-state-vector.py`: Computing the fidelity of a a noisy state vector.
